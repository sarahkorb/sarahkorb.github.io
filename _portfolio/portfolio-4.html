--- title: "Keyboard to Morse Code Translator" excerpt: "Translates keyboard input to Morse code using a FPGA to display message
" collection: portfolio --- Dartmeet is a location-centered social media platform that displays students and alumni. Inspired by the difficulties finding connections during the pandemic, Dartmeet allowed Dartmouth students and alumni to find eachother on and off campus, pursue friendhsips, and share interests and contact information. Dartmeet was designed using Figma and built with React Native. This was a project for a Human Centered Design class.
