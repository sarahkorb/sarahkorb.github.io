--- title: "Keyboard to Morse Code Translator" excerpt: "Translates keyboard input to Morse code using a FPGA to display message
" collection: portfolio --- This Morse code translator was a project for a Digital Electronics course. The user can input a message on the computer keyboard and then the message is displayed in Morse code via a flashing light on the FPGA. Each ASCII encoded character from the message is received as a serial stream, stored in a first-in-first-out queue, and then a lookup table is used to output the correct morse code. More details on implementation can be found in the project report below.
